# OW Coach AI

Hybrid deterministic + generative coaching system for Overwatch.

## What this demonstrates
- Deterministic event modeling (source of truth)
- Bounded LLM reasoning over structured evidence
- Guardrails: citations + abstention + verifier
- Outcome metrics (e.g., first-death rate, death-with-ult)
